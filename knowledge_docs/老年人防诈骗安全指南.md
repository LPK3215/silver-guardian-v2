# 老年人防诈骗安全指南

## 一、常见诈骗类型

### 1. 保健品诈骗
- **套路**：以"免费体检""健康讲座"为名，夸大产品功效，高价推销保健品。
- **特征**：声称"包治百病""祖传秘方""专利产品"。
- **防范**：保健品不能替代药品，购买前咨询医生，到正规渠道购买。

### 2. 冒充公检法诈骗
- **套路**：冒充公安、检察院、法院工作人员，称老人涉嫌犯罪，要求转账到"安全账户"。
- **特征**：要求保密、不准挂电话、要求转账。
- **防范**：公检法不会电话办案，更没有"安全账户"。挂断电话，拨打110核实。

### 3. 冒充亲友诈骗
- **套路**：冒充老人的子女或亲友，称出事故、生病需紧急用钱。
- **特征**：催促紧急转账，不让核实。
- **防范**：先挂电话，用已知号码联系本人核实。设家庭暗号。

### 4. 投资理财诈骗
- **套路**：以"高收益、零风险"为诱饵，推销虚假理财产品。
- **特征**：承诺高额回报，催促"名额有限"。
- **防范**：天上不会掉馅饼。正规理财产品在银行或证券公司购买。

### 5. 中奖诈骗
- **套路**：电话或短信通知中奖，要求先交"手续费""税费"。
- **特征**：未参加却有"中奖"，要求先汇款。
- **防范**：不要相信天上掉馅饼，不汇款、不透露个人信息。

### 6. "以房养老"诈骗
- **套路**：诱导老人抵押房产投资，最终房产被过户。
- **特征**：签复杂的合同，承诺高额回报。
- **防范**：涉及房产的重大决定，务必咨询子女和律师。

## 二、防骗六不三问

### 六不
1. **不轻信**——不轻信来历不明的电话和短信。
2. **不透露**——不透露自己及家人的身份信息、存款、银行卡。
3. **不点击**——不点击陌生链接，不扫描陌生二维码。
4. **不转账**——绝不向陌生账户转账汇款。
5. **不贪小**——不贪图小便宜，不参加"免费"活动。
6. **不独行**——重大财务决定与家人商量。

### 三问
1. 遇到转账要求，先问自己：**认识对方吗？**
2. 遇到"紧急情况"，先问家人：**是真的吗？**
3. 遇到"高额回报"，先问银行：**合理吗？**

## 三、遭遇诈骗后怎么办

1. **立即报警**：拨打110或到就近派出所报案。
2. **保留证据**：保存通话记录、短信、转账凭证等。
3. **冻结账户**：联系银行冻结被骗账户。
4. **告知家人**：不要因为面子隐瞒，及时告知家人协助处理。
5. **心理疏导**：被骗不是老人的错，不要自责，必要时寻求心理帮助。

## 四、居家安全须知

### 用电安全
- 不使用老化、破损的电线和插座。
- 大功率电器不与其他电器共用插座。
- 离开家时关闭电器电源。

### 用火安全
- 厨房用火时不离人。
- 燃气使用后关闭阀门。
- 不在床上吸烟。
- 家中配备灭火器，学会使用方法。

### 防滑防摔
- 卫生间铺设防滑垫，安装扶手。
- 保持地面干燥，及时擦干水渍。
- 走道畅通，不堆放杂物。
- 穿合脚防滑的鞋子。

### 应急准备
- 在家中显眼位置张贴紧急联系电话（家人、120、110、社区）。
- 随身携带紧急联系卡（姓名、疾病史、联系人电话）。
- 独居老人与邻居保持联系，约定每日平安信号。
