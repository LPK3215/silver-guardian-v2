# 护理计划生成技能

当用户需要为老年人制定个性化护理计划时，使用此技能。

## 需要收集的信息

1. **基本信息**：年龄、性别、身高体重
2. **健康状况**：主要疾病诊断、用药情况、过敏史
3. **自理能力**：ADL 评分或自理程度描述
4. **认知状况**：是否认知障碍及程度
5. **居住情况**：独居/与家人/机构，居家环境
6. **照护资源**：主要照护者、可获得的医疗资源

## 护理计划模板

### 一、日常护理安排

| 时间 | 护理内容 | 注意事项 |
|------|---------|---------|
| 06:30 | 起床、如厕、洗漱 | 起床遵循"三个30秒" |
| 07:00 | 早餐 + 晨间用药 | 餐后30分钟服药 |
| 08:00 | 晨间活动/锻炼 | 根据体力安排 |
| 10:00 | 加餐（水果/牛奶） | 糖尿病患者控糖 |
| 11:30 | 午餐 + 午间用药 | 细嚼慢咽 |
| 12:30 | 午休 | 不超过1小时 |
| 14:00 | 下午活动/社交 | 鼓励参与 |
| 15:30 | 加餐 | |
| 17:30 | 晚餐 + 晚间用药 | 清淡为主 |
| 19:00 | 自由活动/看电视 | |
| 20:30 | 睡前准备 | 检查环境安全 |
| 21:00 | 睡前用药 | |

### 二、饮食方案
- 每日热量目标
- 营养素分配比例
- 食物禁忌
- 饮水计划

### 三、运动方案
- 运动类型选择
- 运动强度和时长
- 安全注意事项
- 运动禁忌

### 四、用药管理
- 用药时间表
- 药物副作用观察
- 用药注意事项

### 五、安全防护
- 跌倒预防措施
- 居家环境改造建议
- 紧急联系方式
- 急救预案

### 六、定期复查
- 需要监测的指标
- 复查频率建议
- 就医提醒

## 操作流程

1. 收集老人的基本信息和健康状况。
2. 评估护理需求（可结合 elderly-assessment 技能）。
3. 按模板生成个性化护理计划。
4. 根据特定疾病（如糖尿病、高血压、失智等）调整计划。
5. 标注重点注意事项和风险提醒。
6. 如用户需要，生成可打印的护理计划表。

## 关键约束

- 护理计划不能替代医生的专业建议。
- 用药相关内容必须以医嘱为准。
- 对于特殊疾病（如心衰、肾衰等），需注明饮食和活动的特殊限制。
- 护理计划应定期根据老人状况变化进行更新。
