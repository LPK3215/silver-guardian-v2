# 老年人常见慢性病日常管理指南

## 一、高血压

### 日常监测
- 每日定时测量血压，建议早晚各一次，并做好记录。
- 测量前静坐5分钟，避免饮咖啡、吸烟后测量。
- 目标血压：一般老年人 < 150/90 mmHg；能耐受者可降至 < 140/90 mmHg。

### 饮食管理
- 限盐：每日食盐不超过5克（约一啤酒瓶盖）。
- 多吃新鲜蔬菜水果，增加钾的摄入。
- 减少高脂肪、高胆固醇食物摄入。
- 戒酒或限酒。

### 运动建议
- 适合运动：散步、太极拳、广场舞、游泳。
- 每次运动30分钟左右，每周5次以上。
- 运动强度以微微出汗、不感到疲劳为宜。
- 避免剧烈运动和憋气动作。

### 用药注意
- 按时服药，不可自行停药或加减剂量。
- 注意药物副作用：如服用利尿剂注意补钾。
- 定期复查肝肾功能和电解质。

### 紧急情况识别
出现以下情况立即就医：
- 突然剧烈头痛、呕吐
- 胸闷胸痛
- 视物模糊
- 肢体麻木或无力
- 说话不清

## 二、糖尿病

### 血糖监测
- 空腹血糖目标：4.4-7.0 mmol/L。
- 餐后2小时血糖目标：< 10.0 mmol/L。
- 糖化血红蛋白（HbA1c）目标：一般 < 7.0%。

### 饮食管理
- 主食定量，粗细搭配，粗粮占1/3以上。
- 每餐蔬菜不少于200克。
- 水果在两餐之间食用，选择低糖水果。
- 避免含糖饮料、糕点、蜂蜜等。

### 运动建议
- 餐后1小时运动有助于降血糖。
- 运动时随身携带糖果，防止低血糖。
- 运动前检查足部，穿合适的鞋袜。

### 用药注意
- 口服降糖药按时服用。
- 注射胰岛素者注意轮换注射部位。
- 警惕低血糖：出冷汗、心慌、手抖时立即补充糖分。

### 并发症预防
- 每年检查眼底一次。
- 定期检查足部皮肤，预防糖尿病足。
- 注意尿蛋白检查，早期发现糖尿病肾病。

## 三、冠心病

### 日常注意
- 随身携带急救药物（如硝酸甘油）。
- 避免情绪激动和过度劳累。
- 注意保暖，避免寒冷刺激诱发心绞痛。
- 保持大便通畅，避免用力排便。

### 心绞痛发作处理
1. 立即停止活动，坐下或半卧位休息。
2. 舌下含服硝酸甘油一片。
3. 如5分钟后不缓解，再含服一片。
4. 如连续含服3片仍不缓解，立即拨打120。

### 饮食建议
- 低盐低脂饮食。
- 每日食盐 < 5克，食用油 < 25克。
- 多吃鱼类、坚果、全谷物。
- 避免动物内脏、肥肉、油炸食品。

## 四、骨质疏松

### 预防措施
- 每日补充钙质：牛奶300ml + 豆制品 + 深绿色蔬菜。
- 适当晒太阳，每日15-30分钟，促进维生素D合成。
- 适度负重运动：散步、慢跑、太极拳。
- 预防跌倒：卫生间安装扶手，地面保持干燥，穿防滑鞋。

### 用药建议
- 钙剂和维生素D补充需遵医嘱。
- 双膦酸盐类药物需空腹服用，服后保持直立30分钟。
- 定期进行骨密度检查。

## 五、老年用药安全

### 用药原则
- 少用药：能用非药物方法解决的不用药。
- 用药精简：定期整理药箱，停用不必要的药物。
- 遵医嘱：不自行增减剂量，不轻信偏方。

### 注意事项
- 建立用药清单，记录所有正在服用的药物。
- 注意药物间相互作用，多种药物同时服用时咨询药师。
- 出现新的不适时，先考虑是否为药物副作用。
- 药物存放注意避光、防潮，定期清理过期药品。
