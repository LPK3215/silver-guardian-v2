# 养老评估技能

当用户需要评估老年人的综合健康状况、生成评估表单或进行风险筛查时，使用此技能。

## 评估维度

### 1. 日常生活能力评估（ADL）
使用 Barthel 指数评估，包含 10 个项目：
- 进食、洗澡、修饰、穿衣、控制大便、控制小便
- 如厕、床椅转移、平地行走、上下楼梯

评分标准：
- 0-20分：完全依赖
- 21-40分：严重依赖
- 41-60分：中度依赖
- 61-80分：轻度依赖
- 81-100分：完全自理

### 2. 认知功能筛查（MMSE 简版）
- 定向力：今天几号？星期几？我们在哪里？
- 记忆力：记住三个词语（如苹果、桌子、硬币），5分钟后复述
- 计算力：100减7，连续减5次
- 语言能力：复述一句话、按要求做动作

评分标准：
- 27-30分：正常
- 21-26分：轻度认知障碍
- 10-20分：中度认知障碍
- <10分：重度认知障碍

### 3. 跌倒风险评估
评估因素：
- 过去一年是否有跌倒史
- 是否使用助行器具
- 是否服用镇静/降压药物
- 视力是否正常
- 步态是否稳定
- 居住环境是否有跌倒隐患

风险等级：低风险（0-1项）、中风险（2-3项）、高风险（≥4项）

### 4. 营养状况评估（MNA 简版）
- 近三个月体重是否下降
- 进食量是否减少
- BMI 是否低于 23
- 是否有急性疾病或应激
- 活动能力如何
- 神经心理问题

### 5. 情绪状态评估（GDS-5 简版）
5个问题：
1. 你对自己的生活基本满意吗？
2. 你是否感到空虚？
3. 你是否害怕不幸发生在你身上？
4. 你是否感到快乐？
5. 你是否经常感到无力回天？

≥3个负面回答提示可能存在抑郁倾向，建议进一步评估。

## 操作流程

1. 确认评估对象的基本信息（年龄、性别、主要健康状况）。
2. 根据用户需求选择评估维度（全面评估或单项评估）。
3. 逐项询问评估问题，记录回答。
4. 计算评分，给出评估结果和等级。
5. 根据评估结果给出建议（如建议就医、改善居家环境、调整饮食等）。
6. 如用户需要，生成可打印的评估报告。

## 关键约束

- 评估结果仅供参考，不能替代专业医疗诊断。
- 认知功能评估和情绪状态评估需要由专业人员解读。
- 对于评估中发现的高风险项，应明确建议就医或寻求专业帮助。
- 评估过程中注意尊重老人隐私和尊严。
